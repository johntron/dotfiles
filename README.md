# dotfiles
John Syrinek's dotfiles

## Installation
```sh
    ./install.sh
```

## Usage
vim:
* Type `gf` to go to file/module under cursor
* Type `:Nedit module` to edit module's index.js
* Type `html:5` then ctrl-Y,comma to expand to HTML5 boilerplate
